# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-11-01

### Added
- Initial release of Thumbnail Generator
- Batch image processing with automatic directory scanning
- Support for PNG, JPG, JPEG, BMP, GIF formats
- High-quality bicubic interpolation resizing
- Automatic duplicate detection and skipping
- Configurable thumbnail size and JPEG quality
- Windows executable compilation support
- Comprehensive error handling and logging

### Features
- 200x200px thumbnail generation (maintains aspect ratio)
- `_thumbnail` suffix naming convention
- Real-time progress reporting
- Cross-directory recursive processing
- No external dependencies (uses .NET Framework)
